# RegexLab

A regex testing and explanation tool built with Vite.

## Features

- **Real-time matching** — See matches highlight as you type
- **Flag toggles** — Global (g), case-insensitive (i), multiline (m), dotAll (s), unicode (u)
- **Match details** — View match text, indices, and captured groups
- **Regex Explainer** — See what each part of your pattern does in plain English
- **Copy explanation** — Export the explanation to share or save

## Quick Start

```bash
npm install
npm run dev
```

Then open `http://localhost:5173/`

## Build

```bash
npm run build
```

Production files are in the `dist/` folder.
